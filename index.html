{CSS}

Cascading Style Sheets

CSS означает каскадные таблицы стилей и представляет собой язык разметки стилей (как HTML или XML).
Таким образом CSS ничего не представляет сам по себе, если не связан с HTML-документом.
CSS привносит жизнь в HTML-документ, выбирая шрифты, применяя цвета, определяя отступы, позиционируя элементы, анимируя взаимодействия и многое другое.

Синтаксис CSS

У языка CSS относительно простой синтаксис.
CSS представляет собой процесс из трёх частей:
- селектор определяет, на какой элемент или элементы нацелиться;
- свойство определяет характеристики для изменения;
- значение определяет, как изменить эту характеристику.

Сначала прописывается селектор. Потом, после фигурных скобок, указываются свойства со значениями — между ними ставится двоеточие.
Сами свойства отделяются друг от друга точкой с запятой.
Весь этот блок (селектор, свойство, значение) является правилом CSS.

CSS правило
селектор { свойство: значение; }

Как работает CSS?

CSS выбирает элемент HTML (например, абзац), задаёт свойство для изменения (такое как цвет) и применяет определенное значение (например, красный):

р { color: red; }

! Можно подумать, CSS используется только для изменения цвета текста, размера и типа шрифта.
Но CSS может задать компоновку HTML-документа, определяя высоту, ширину, внутренние и внешние поля, положение, колонки и др.

{CSS селекторы}

Селектор- это мощнейший инструмент идентификации определенных элементов страницы или же группы элементов объединенных конкретным признаком для дальнейшего применения к ним стилей оформления из блока объявлений относящегося к данному селектору.
Поскольку мы не хотим стилизовать все наши элементы HTML одновременно, нам требуется возможность выбрать подмножество этих элементов.
Так, именно благодаря селекторам определяем к каким элементам мы хотим применить стиль.

! Обратите внимание - не существует селекторов, которые бы позволили выбрать родителя (содержащий контейнер) или соседа родителя или потомков соседа родителя.

Базовые селекторы

Селектор по тегу (элементу)

Этот базовый селектор предназначен для выбора элементов по имени тега.

Синтаксис:имяТега

Пример использования селектора по тегу:

/* селектор h1 выберет все элементы h1 на странице */

h1 {
 color: red;
}

Селектор по классу

Селекторы элементов определяют стиль сразу всех выбранных элементов, тогда как для сайтов часто требуется задать стиль отдельных элементов.
Этот базовый селектор предназначен для выбора элементов по классу (значению атрибута class).

Синтаксис:  .имяКласса

Пример использования селектора по классу:

/* селектор text выберет все элементы text на странице */

.text {
 color: blue;
}

Селектор по идентификатору

Идентификатор определяет уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты.
Этот базовый селектор предназначен для выбора элементов по идентификатору (значению атрибута id). Может быть применен только к одному элементу.

Синтаксис:  #имяИдентификатора

Пример использования селектора по идентификатору:

/* селектор #text-1 выберет элемент, имеющий идентификатор text-1 */

#text-1{
 color: purple;
}

Селектор по атрибуту

Этот базовый селектор предназначен для выбора элементов по имени атрибута и (или) его значению.

Синтаксис:  атрибут {}

Пример использования селектора по атрибуту:

/* универсальный селектор выберет все элементы на странице */

input[type="text"]{
 border: 2px solid red;
}

Универсальный селектор

Этот базовый селектор предназначен для выбора всех элементов.

Синтаксис:*

Пример использования универсального селектора:

/* универсальный селектор выберет все элементы на странице */

*{
 box-sizing: border-box;
}

Комбинаторы

Комбинаторы, как следует из названия, предназначены для того чтобы комбинировать, объединять различные селекторы.
Существует четыре типа комбинаторов для указания отношений между элементами: Комбинатор потомков (пробел)
Комбинатор дочерний (>)
Комбинатор соседний родственный (+)
Комбинатор общий родственный (~)

Рассмотрим каждый из них:

Комбинатор потомков

Комбинатор потомков соответствует элементу В, который является потомком элемента А.
Обратите внимание, что комбинатор потомков выбирает всех потомков.

Синтаксис: А В

Пример использования комбинатора потомков:

/* комбинатор потомков выберет все элементы на странице */

ul li {background: black;}

Комбинатор дочерний

Дочерний селектор выбирает все элементы, которые являются потомками указанного элемента.
Будут выбраны только те элементы, соответствующие второму селектору, которые являются прямыми потомками элементов, соответствующих первому селектору.
Все элементы-потомки на более низких уровнях иерархии будут пропущены.

Синтаксис: А>В

Пример использования дочернего комбинатора:


ul > li {
 text-decoration: underline;
}

Комбинатор соседний родственный

Комбинатор соседний родственный выбирает только элемент, который следует сразу после другого элемента.

Синтаксис: А+В

Пример использования соседнего родственного комбинатора:


div + h1{
 background-color: black;
}

Комбинатор общий родственный

Комбинатор общий родственный выбирает все элементы, которые являются братьями и сестрами указанного элемента.

Синтаксис: А~В

Пример использования общего родственного комбинатора:


div ~ h1{
 background-color: black;
}

Псевдоклассы и псевдоэлементы

Псевдоклассы – это селекторы, которые определяют состояние уже существующих элементов, которое может меняться при определенных условиях.
Псевдоэлементы – это селекторы, которые определяют область элементов, которая изначально отсутствует в дереве документа. Эта область создается искусственно с помощью CSS.

Псевдоклассы

Существует достаточно большой выбор псевдоклассов.
Элементы HTML могут иметь разные состояния. Наиболее распространенный случай- когда вы наводите курсор на ссылку.
В CSS возможно применить другой стиль, когда происходит такое событие.
Псевдоклассы привязаны к обычным селекторам и начинаются с двоеточия:

Примеры псевдоклассов:

E:link E:target E:root E:root E:last-child
E:visited E:lang(fr) E:nth-child(n) E:nth-child(n) E:first-of-type
E:active E:enabled E:nth-last-child(n) E:nth-last-child(n) E:last-of-type
E:hover E:disabled E:nth-of-type(n) E:nth-of-type(n) E:only-child
E:focus E:checked E:nth-last-of-type(n) E:nth-last-of-type(n) E:only-of-type
E:target E:not(s) E:first-child E:first-child E:empty

Применение псевдокласса: focus

<!DOCTYPE HTML>
<html>
<head>
<meta charset="utf-8">
<title>Псевдоклассы</title>
<style>
INPUT:focus {
color: red; /* Красный цвет текста */
}
</style>
</head>
<body>
<form action="">
<p><input type="text" value="Черный текст"></p>
<p><input type="text" value="Черный текст"></p>
</form>
</body>
</html>

Псевдоэлементы

Псевдоэлементы позволяют задать стиль элементов не определённых в дереве элементов документа, а также генерировать содержимое, которого нет в исходном коде текста.
Псевдоэлементы начинаются с ::, чтобы отличить их от псевдоклассов.

Примеры псевдоэлементов:


E::first-line
E::first-letter
E::before
E::after

Применение псевдоэлемента: after

<!DOCTYPE HTML>
<html>
<head>
<meta charset="utf-8">
<title>Псевдоэлементы</title>
<style>
P.new:after {
content: " - Новьё!"; /* Добавляем после текста абзаца */
}
</style>
</head>
<body>
<p class="new">Ловля льва в пустыне с помощью метода золотого сечения</p>
<p>Метод ловли льва простым перебором.</p>
</body>
</html>
